游玩前请注意

服务器目前只支持 Java 版 1.21 版本, 如有不便之处, 请谅解😯

我们并不要求你使用正版 MC 进行游玩, 你可以使用 PCL/HMCL 等

但当你使用正版账号游玩时, 你也可以享用使用正版功能(皮肤/披风/自定义名称等...)

服务器正在准备升级至 1.21.3...(没钱)


为维护良好的游玩环境，本服务器不希望 熊孩子、作弊者、炸图者 等有不良行为的玩家游玩!!!

如果你发现有玩家有不良行为，请向 管理员 或 服主腐竹 举报
